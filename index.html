The matrix

Instruction

The matrix

The thing is there are 3 different data M (Rows) - response for the number of rows, N (Columns) - response for the number of columns and X (Closets) - response for the number of approximate numbers that will be highlighted when you hover with the mouse on a number from the table.

Суть в том что на вход есть 3 разных данных M (Rows) - отвечает за количество строк, N (Columns) - отвечает за количество столбцов и X (Closets) - отвечает за количество приближенных чисел которые будут подсвечиваться когда вы делаете hover мышкой на число из таблицы.

If you change the data and click on the 'Create matrix' or 'Enter' button, a new matrix with your parameters will be created. There is also a column 'Sum:' which shows the sum of one row, with a hover effect on the amount, each amount will change by a percentage of the total amount and on each percentage there will be a background depending on the percentage. When you click on a number from the table, its sum will increase by 1, as well as the total amount of the row and the average value for the column (the 'Average in row' line with a purple background). With using the 'Delete' button you can delete a line, and using the '+' add a line.

Если вы измените данные и нажмите на кнопку 'Create matrix', или 'Enter' то создастся новая матрица с вашими параметрами. Также присутствует столбец 'Sum:'' в котором показана сумма одного ряда, при hover эффекте на сумму каждая сумма изменится на процент от общей суммы и на каждом процентные будет фон в зависимости от процентов. При клике на число из таблицы его сумма увеличится на 1, а также увеличится общая сумма по строке и среднее значение по столбцу (строка 'Average in row' с фиолетовым фоном). Ну и с помощью кнопки 'Delete' вы можете удалить строку, а с помощью '+' добавить строку.
